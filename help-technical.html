Page Not Loading

1
Refresh the page — Press F5 or Ctrl+R (Cmd+R on Mac).
2
Check your internet connection — Open another website to confirm you're online.
3
Clear your browser cache — Press Ctrl+Shift+Delete (Cmd+Shift+Delete on Mac) → select Cached images and files → Clear data.
4
Try a different browser — If the page loads in another browser, the issue is specific to your current browser's settings.
5
Disable browser extensions — Ad blockers or privacy tools can sometimes block scripts. Try in an Incognito/Private window.
If the problem persists, contact us with the URL that isn't loading and the error message you see.

Login / Sign-in Errors

1
"Wrong email or password" — Double-check your email address (no typos, no spaces). Use Forgot Password if needed.
2
"Please confirm your email" — Check your inbox (and spam) for the confirmation link sent when you registered.
3
Google/Discord sign-in not working — Make sure pop-ups aren't blocked in your browser. Try in a different browser or Incognito mode.
4
"Too many attempts" — Wait 1–2 minutes before trying again. Your account is temporarily rate-limited for security.

Chapter Not Loading / Blank Page

1
Refresh the page — most blank chapter issues resolve on refresh.
2
Check that you're connected to the internet and try navigating to another page first.
3
The chapter may have been taken down by the author or our moderation team. Return to the novel's chapter list to check.
If a specific chapter consistently fails to load, please contact us with the chapter URL.

Mobile Issues

1
Text too small or broken layout — Make sure your browser zoom is set to 100%. In Chrome on Android: tap ⋮ → Settings → Accessibility → Text Scaling.
2
Buttons not responding — Try a full page reload. If using an in-app browser (e.g. Instagram), open in Chrome or Safari instead.
3
Install as app — In Chrome tap ⋮ → Add to Home Screen. This gives you a better experience than the browser tab.

Covers / Images Not Showing

1
Reload the page — image loading delays are usually temporary.
2
Check if your browser is blocking mixed content or images. Chrome: click the padlock icon → Site Settings → Images → Allow.
3
If you're on a slow connection, images may take longer to appear. Wait a few seconds.

Frequently Asked Questions

The site looks broken / no CSS styles. ▾
Clear your browser cache (Ctrl+Shift+Delete) and reload. If the issue persists, try a hard refresh: Ctrl+Shift+R on Windows/Linux, Cmd+Shift+R on Mac.
My profile picture won't upload. ▾
Make sure the file is JPG, PNG or WEBP and under 2MB. If it still fails, try a different image or contact support.
I'm seeing an error message I don't understand. ▾
Take a screenshot of the error and contact us — include the page URL and the full error message.
The site is very slow. ▾
Try on a different network. Clear your browser cache. Disable browser extensions. If you're on mobile, try Wi-Fi instead of mobile data.
A feature that used to work has stopped working. ▾
It may be a temporary issue. Refresh and try again. If the problem persists for more than a few hours, please contact us so we can investigate.
